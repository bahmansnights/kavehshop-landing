# 🛍️ KavehShop Landing Page (Inspired Redesign)

This is a custom-built **landing page** created with HTML, CSS, and JavaScript, inspired by a Persian WordPress theme homepage design.  
![kaveh-pcc](https://github.com/user-attachments/assets/d19e5892-dd81-4da1-a9ae-7682218e6b27)

You can preview it live here:  
🔗 [Live Demo](https://kavehshopdevbybahman.netlify.app)

## ✨ Features

- Fully responsive
- Clean modern layout
- Swiper.js slider integration
- No frameworks — just HTML + CSS + JS
- Optimized for performance

## 🎯 About the Project

This project was built from scratch as a **practice** and **showcase** of front-end development skills.  
The visual style was inspired by an existing Persian WordPress theme, but no source code or assets were reused.

## 📦 Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Responsive Media Queries)
- JavaScript (Vanilla)
- [Swiper.js](https://swiperjs.com/) (for the image/hero slider)
- Netlify (for deployment)
